# Flask-TUI-Editor

[Toast UI Markdown Editor](https://ui.toast.com/tui-editor) Integration for Flask-WTF.

## Installation
To install Flask-TUI-Editor:

```
pip install flask-tui-editor
```

Check [example](example/) project and documentation.
